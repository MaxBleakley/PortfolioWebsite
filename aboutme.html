Projects

I have participated in several projects, one of which was the configuration of a LAN network using enterprise equipment including Cisco Catalyst network switches allowing for basic file sharing capabilities between devices. Additionally, I configured a basic PiHole network adblock using a Raspberry Pi 4 to filter traffic entering and exiting my home network on all devices including smart TV's etc. This Project was later discontinued as I repurposed it into a server using enginx to host a website I had created to demonstrate the functionality of a password manager I developed

The next notable project i created was a 2D RPG game which drew heavy inspiration from games such as Stardew Valley. This game was created in the unity game engine and primarily programmed in C sharp and had features such as inventory management (including drag and drop functionality), monetary/shop systems, crop growing/planting mechanics and NPC's with interactable dialogue among other features.

Finally, I worked with a fellow student at SERC to configure over 100 lenovo tablets working for the local city council to aid them in an initiative to help connect vulnerable and those who are socially isolated. This included the creation of detailed, yet non-technical guidance documents, assesment of user needs and communication, primarily with the local council who requested our assistance.
